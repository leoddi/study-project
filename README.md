# Hotel Fischer Homepage - study rebuild

A personal study project recreating the homepage of the Fischer Hotel in Bressanone made using NextJs and Tailwind CSS. The orginal website was build using different frameworks. I focused mainly on creating an accurate design, loyal to the original. The website has been kept responsive across all device sizes and accesibility basics are implemented.

## Built using

- Nextjs
- Tailwind CSS
- Deployed on aws amplify
